# Rust No BS Crash Course Exercises

## Getting started

- Make sure you're on the `master` branch `git checkout master`
- Find exercise related to the video you just watched: `src/vid_i.rs` where `i` = video number
- Implement your solution(s). Do not edit the test suite.
- Check your solution(s) with `cargo test`
- Compare your solution(s) with mine. Either pull up `solutions` branch on GitHub or `git diff solutions -- src/vid_i.rs` where `i` = video number
